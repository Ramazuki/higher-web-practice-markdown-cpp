# MarkdownToHTML Converter

CLI-приложение для конвертации упрощённого Markdown в HTML.

## Возможности

Поддерживаемые конструкции Markdown:

- **Заголовки** — `#`, `##`, `###`
- **Параграфы** — несколько строк склеиваются в один абзац
- **Нумерованные списки** — `1. ...`, `2. ...`
- **Маркированные списки** — `- ...` или `* ...`
- **Жирный текст** — `**bold**`
- **Курсив** — `*italic*`
- **Инлайн-код** — `` `code` ``
- **Ссылки** — `[текст](url)`
- **Экранирование** — `\*`, `\<`, `\>` и другие спецсимволы
- **HTML-экранирование** — `&`, `<`, `>`, `"` в выводе

Непарные маркеры (`*`, `**`, `` ` ``, `[`) трактуются как обычный текст.

## Сборка

Требования: `cmake` >= 3.30, компилятор C++20.

```bash
cmake -B build -S .
cmake --build build --parallel
```

Или одной командой:

```bash
./build.sh
```

## Использование

Вывод в stdout:

```bash
./build/MarkdownToHTML --in examples/input.md
```

Запись в файл:

```bash
./build/MarkdownToHTML --in examples/input.md --out result.html
```

### Пример

Вход (`examples/input.md`):

```markdown
# Заголовок

Текст с *курсивом*, **жирным** и `кодом`.

- Пункт A
- Пункт B

Ссылка: [поиск](https://example.org)

Экранирование: \*звёздочки\* и \<скобки\>.
```

Выход:

```html
<h1>Заголовок</h1>
<p>Текст с <em>курсивом</em>, <strong>жирным</strong> и <code>кодом</code>.</p>
<ul>
  <li>Пункт A</li>
  <li>Пункт B</li>
</ul>
<p>Ссылка: <a href="https://example.org">поиск</a></p>
<p>Экранирование: *звёздочки* и &lt;скобки&gt;.</p>
```

## Тестирование

Запуск всех тестов:

```bash
./run_tests.sh
```

Только юнит-тесты:

```bash
cmake --build build --parallel
./build/MarkdownTests
```
